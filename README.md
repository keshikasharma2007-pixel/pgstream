# pgstream – PostgreSQL CDC API

## Overview
pgstream is a Python REST API that simplifies **PostgreSQL Change Data Capture (CDC)** by managing logical replication (publications & subscriptions) programmatically.

It enables developers to track and stream database changes without manually configuring PostgreSQL replication.

---

## Features
- Create/manage publications and subscriptions via API  
- Secure SQL execution (prevents SQL injection)  
- Config-driven setup  
- Structured logging  
- Clean layered architecture  

---

## Tech Stack
- Python, PostgreSQL, psycopg2  
- Docker, Postman, DBeaver
